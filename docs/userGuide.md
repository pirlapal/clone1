# User Guide

**Please ensure the application is deployed, instructions in the deployment guide here:**

Once you have deployed the solution, the following user guide will help you navigate the functions available.

- The application provides intelligent multi-domain conversational AI assistance for Tuberculosis (TB), Agriculture, and General health/education topics through natural language interaction with specialized agents that automatically route queries to the most appropriate expert.
  
1. Users can ask questions through the chat interface and get expert responses with citations from the knowledge base.
   ![Chat Interface](../docs/media/user-interface.gif)
   - The main chat interface with real-time streaming responses and conversation history.

   ![yet to be put placeholder](../docs/media/quick-start-questions.png)
   - Quick start suggestions including TB questions ("What causes TB?", "What are the main symptoms of pulmonary TB?", "What is Ni-Kshay used for?") and Agriculture questions ("How to improve crop irrigation efficiency?", "What are sustainable farming practices?", "What is crop rotation?").
  
2. The system supports both text and image inputs, automatically routing to appropriate specialist agents (TB, Agriculture, or General) based on query content.
